Detailed Syllabus

1. Big Data Mindset

  • Evolution of data systems (RDBMS → Lakehouse)
  • Big Data mindset & industry shifts
  • Modern Data Engineering roles

2. Python for Data Engineering

  • Python syntax, control flow, functions, file I/O, string/date handling
  • Pandas, reading/writing files
  • Error handling

3. SQL for Data Engineering

  • SELECT, WHERE, GROUP BY, HAVING, JOINs, subqueries, CTEs
  • Window functions, CASE/WHEN, PIVOT, nested queries
  • SQL Project
  • Delta Table components, key features & architecture

4. Spark Foundations

  • Hadoop vs Spark
  • RDD vs DataFrame
  • Lazy evaluation, transformations & actions
  • Spark APIs overview

5. Databricks Lakehouse Architecture

  • Lake vs warehouse vs lakehouse
  • Control vs data plane, clusters, runtime versions
  • Notebooks, multi-language support
  • Repos, GitOps, notebook sharing & execution

6. Data Ingestion with Lakeflow

  • Lakeflow Connect concepts
  • Batch, incremental batch & streaming batch
  • Enterprise data ingestion approaches

7. Declarative Pipelines: Data Transformation & Validation

  • What are declarative pipelines & how they work
  • Dataset types
  • Streaming & materialized view overview
  • Temporary views, compute
  • Data quality & expectations

8. Databricks Performance Optimization & Unity Catalog

  • Spark architecture & foundation design
  • Code optimization & cluster tuning
  • Metastores, catalogs, schemas, tables
  • Permissions & object hierarchy

9. Gen AI Basics on Databricks

  • Introduction to Gen AI
  • Prompt engineering techniques

10. Capstone Project Implementation

  • Bronze/Silver/Gold pipeline creation
  • Transformations & quality checks

11. Mock Exam & Feedback

  • Full-length mock test
  • Discussion & exam tips
  • Open Q&A session
